# DLH Siplah Terpadu

<AnimatedName />

![DLH Siplah Terpadu Landing Page Overview](/projects/siplah-terpadu/first.png)

DLH ([Dinas Lingkungan Hidup](https://www.instagram.com/dlh.ppu/)) Siplah Terpadu is a centralized public service and complaint reporting platform developed for the Environment Agency (DLH) of Penajam Paser Utara to streamline reporting workflows, permit applications, and administrative oversight.

![DLH Siplah Terpadu Admin Dashboard and Reporting Workflows](/projects/siplah-terpadu/second.png)

## Main Features

- **Centralized Reporting Platform**: Streamlines the process for public reports and permit applications, modernizing service delivery for the Environmental Department.
- **Role-Based Access Control (RBAC)**: Multi-level RBAC featuring distinct dashboards for Users, Admins, and Super Admins to enable secure management of reporting workflows and follow-up actions.
- **Authentication & Document Management**: Integrated NextAuth.js for secure authentication and Firebase Storage for document management, supporting efficient handling of official file attachments.

![DLH Siplah Terpadu Interactive Map & Report Details using Leaflet](/projects/siplah-terpadu/third.png)

## Tech Stacks

React, Next.js (App Router), Tailwind CSS, shadcn/ui, NextAuth.js, Firebase Storage, Prisma ORM, Leaflet.
