# Wedding Invitational Website

<AnimatedName />

![Wedding Invitation Digital Platform Hero Section](/projects/wedding-invitation/first.png)

A high-performance digital invitation platform featuring dynamic guest personalization and interactive engagement systems.

![Wedding Invitation Guest RSVP and RSVP Form](/projects/wedding-invitation/second.png)

## Main Features

- **Dynamic Personalization:** Features an invitation link generator that automates guest personalization via URL parameters `?to=GuestName`, supporting real-time previews and cultural templates.
- **Interactive Engagement:** Integrates a Guest Wish system with CRUD operations and input validation, connected with the WhatsApp API and Web Share API for seamless distribution.
- **Modern User Experience:** Utilizes Tailwind CSS 4 for a responsive, mobile-first design that prioritizes accessibility and performance.

![Wedding Invitation Guest Wishes & RSVP Comments Section](/projects/wedding-invitation/third.png)

## Tech Stacks

Laravel 13, Inertia.js, React 19, Tailwind CSS 4, WhatsApp API, Web Share API.
