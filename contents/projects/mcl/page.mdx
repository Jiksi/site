# Meranti Creative Lab

<AnimatedName />

![Meranti Creative Lab (MCL) Hero Showcase & Branding](/projects/mcl/first.png)

Meranti Creative Lab is a Design & Brand Consultant based in Balikpapan City. Collaborated with the MCL team to develop a responsive landing page and portfolio showcase platform that delivers a modern, high-performance, and interactive user interface.

![Meranti Creative Lab (MCL) Portfolio Gallery & Services](/projects/mcl/second.png)

## Main Features

- **Responsive Landing Page**: Offers a modern, high-performance, and interactive user interface, providing a seamless user experience across devices.
- **Headless CMS Architecture**: Integrates the WordPress API with the Next.js frontend to enable dynamic, efficient management of project content.
- **Performance Optimization**: Optimizes User Experience (UX) and Core Web Vitals via advanced image placeholder techniques using Plaiceholder (Base64), minimizing layout shifts and improving perceived loading speeds.

![Meranti Creative Lab (MCL) Case Study & Project View](/projects/mcl/third.png)

## Tech Stacks

React, Next.js, Tailwind CSS, Framer Motion, WordPress API, Plaiceholder, GraphQL.
