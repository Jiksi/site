# Indolocation

<AnimatedName />

![Indolocation Landing Page and Search Interface](/projects/indolocation/first.png)

Indolocation is a comprehensive location booking and directory platform featuring a sophisticated search engine, multi-role access management, and professional project documentation utilities.

![Indolocation Location Details and Search Filters](/projects/indolocation/second.png)

## Main Features

- **Location Booking & Directory:** Features a search engine with multi-criteria filtering (location, category, facilities) and dynamic sorting to facilitate seamless location search and booking.
- **Multi-Role Access Control:** A secure role management system (Admin, Producer, User) utilizing Spatie Laravel Permission and Google Socialite for seamless authentication and content curation.
- **Documentation & Export Utilities:** A "My-List" collection system and PDF export utility using `html2pdf.js`, enabling users to generate curated reports.

![Indolocation Dashboard and PDF Report Export Features](/projects/indolocation/third.png)

## Tech Stacks

Laravel 11, Inertia.js, React 18, Spatie Laravel Permission, Google Socialite, html2pdf.js, Tailwind CSS.
