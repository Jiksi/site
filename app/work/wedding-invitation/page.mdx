## Main Features

- **Dynamic Personalization:** Features an invitation link generator that automates guest personalization via URL parameters `?to=GuestName`, supporting real-time previews and cultural templates.
- **Interactive Engagement:** Integrates a Guest Wish system with CRUD operations and input validation, connected with the WhatsApp API and Web Share API for seamless distribution.
- **Modern User Experience:** Utilizes Tailwind CSS 4 for a responsive, mobile-first design that prioritizes accessibility and performance.
